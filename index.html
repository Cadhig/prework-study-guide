HTML

  • The head element contains information about the webpage.
  • The body element represents the visible conntent shown to the user
  • Body elements have code written within them called child code
  • h1-h6 : headers
  • p : block of text/paragraph
  • ul : creates an unordered list
  • li : lists items with bullets
  • img src="img.jpg" : embeds an image, can also adjust the size using height="123" and width="123"
  • br : creates a blank line
  • a href="url" : embeds a link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much spacce we want around the content inside an element.
  • width and height can be done in px

GIT

  • git status : checks the branch we are currently on
  • git checkout -b feature/name : creates a new branch
  • git add -A : the "add" command adds modificationns in the current working branch to the staging area, the "-A" flag indicates we want to add all changes
  • git pull origin main : recievves branch modification to local, indicates the source of the pull will be in github repo
  • git push originn feature/name : pushes the changes to the remote branch
